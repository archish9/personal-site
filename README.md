# Senior Developer Portfolio

High-performance portfolio built with Next.js 14 for optimal SEO and fast load times.

## Features

- ✅ Server-side rendering for SEO
- ✅ Pure CSS animations (no JS libraries)
- ✅ Dark theme with subtle gradients
- ✅ Fully responsive
- ✅ Accessible (semantic HTML, keyboard navigation)
- ✅ Fast load time (<1s FCP)

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build for Production

```bash
npm run build
npm start
```

## Customize

Edit `app/page.js` to update:
- Name, role, bio
- Experience details
- Blog links
- Contact information

Edit `app/globals.css` for styling adjustments.

## Performance

- No JavaScript animation libraries
- Minimal bundle size
- Automatic code splitting
- Static generation where possible
